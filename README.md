# zig-gb

![](screenshot.webp)

## Getting started

- [zig](https://ziglang.org/download/)

```
git clone https://github.com/zig-homebrew/zig-gb
cd zig-gb/
zig build # then run zig-out/zig-gba.gb with vbam
```
